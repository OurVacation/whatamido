# 📅 WhatAmIDo: Rust-Powered Calendar Assistant

> **"계획만 세우는 것은 그만, 이제는 달성률까지 관리하세요."**
> `WhatAmIDo`는 Google Calendar와 연동하여 실시간 알림을 보내고, 작업의 진척도를 추적하여 하루의 성과를 시각화해주는 Rust 기반 CLI 도구입니다.

---

## ✨ 핵심 기능 (Features)

* **Google Calendar 동기화**: Google API를 통해 오늘의 일정을 자동으로 가져옵니다.
* **실시간 알림 (Active Notification)**: 일정 시작 시간이 되면 데스크톱 알림과 터미널 메시지로 알려줍니다.
* **진척도 추적 (Progress Tracking)**: 일정이 종료되면 해당 목표를 얼마나 달성했는지(`%`) 입력받아 기록합니다.
* **일일 요약 보고서 (Daily Summary)**: 하루가 끝날 때, 오늘 계획의 전체 달성도와 성패를 한눈에 보여줍니다.
---

## 단계별 목표 달성

1. `chrono` crate를 사용하여 현재 시간을 가져오고, 목표 시간과 비교하여 터미널에 알람을 전송하자! (26/03/03~26/03/04)
	- 핵심 목표 : 일단 터미널에 현재 시간과 계획 시간을 비교해서 알림을 띄워보자
	- 수행 과정
		1. 실행 시 계획명, 시작 시간, 종료 시간을 수동 입력
		2. 입력한 정보를 `Vec<Plan>` 으로 파싱해서 넣음
		3. 현재 시간(`Chrono::Local::now()`)과 위의 `Vec`의 계획 시간 정보를 비교하여 계획 시작, 종료 알림
		4. 진척도를 터미널에 수동 입력 후, Vec의 모든 Plan의 state의 varient가 Done이 되면 진척도 나열 후 종료
	- 과정 중 학습 내용
		1. 계획을 저장하는 Plan 구조체를 plan.rs에 분리 후 mod, use 키워드를 이용해서 main에서 통합했다.
		2. `[#derive(...)]` 의 사용법과 사용하는 이유
		3. plan 구조체 작성시 계획 상태를 나타내는 열거형을 사용했다.
	- 해당 단계 고칠 점
		1. 진척도 입력 시, u8 범위를 벗어난 수를 입력하면 expect로 패닉을 일으켜 버리게 했는데 이건 좋지 않은거 같다. 재입력을 받거나 하는 것이 적절한 방향일 것 같다.
